Documentation de l'API

Bienvenue dans la documentation de l'API ! Voici les endpoints disponibles :

Endpoints pour les modèles

  • POST /api/register - Création d'un compte
  • POST /api/token - Authentification
  • POST /api/login - Authentification plateforme Admin
  • POST /api/manage-users - Gestion des comptes

    • (Authentification et validation requises)

    Models

    GET /api/models - Retrieve a list of car models

    Example response:

    [
    {
        "id": 1,
        "brand": "BMW",
        "name": "Serie 3 e46",
        "year": 2004,
        "price": "31500.00"
    },
    {
        "id": 2,
        "brand": "Audi",
        "name": "A4 B8",
        "year": 2010,
        "price": "27500.00"
    }
]

    Create a Model

    POST /api/create/model - Create a new car model

    Example request:

    {
    "brand": "BMW",
    "name": "Serie 3 e46",
    "year": 2004,
    "price": "31500.00"
}

    Update a Model

    PATCH /api/update/model/{id} - Update a car model

    Example request:

    {
    "brand": "BMW",
    "name": "Serie 3 e461",
    "year": 2003,
    "price": "31000.00"
}

    Delete a Model

    DELETE /api/delete/model/{id} - Delete a car model

    Brands

    GET /api/marques - Retrieve a list of car brands

    Example response:

    [
     {
        "id": 1,
        "group": "Groupe BMW",
        "name": "BMW"
    },
     {
        "id": 1,
        "group": "Groupe Volkswagen",
        "name": "Volkswagen"
    }
]

    Create a Brand

    POST /api/create/marque - Create a new car brand

    Example request:

    {
      "group": "Groupe BMW",
      "name": "MINI"
}

    Update a Brand

    PATCH /api/update/marque/{id} - Update a brand

    Example request:

    {
      "group": "Groupe Volkswagen",
      "name": "Volkswagen"
}

    Delete a Brand

    DELETE /api/delete/marque/{id} - Delete a Brand

    Groups

    GET /api/groupes - Retrieve a list of groups

    Example response:

    
    {
        "name": "Groupe BMW"
    }

    Create Groups

    POST /api/create/groupe - Create a new car group

    Example request:

    
    {
        "name": "Groupe Fiat"
    }

    Update Groups

    PATCH /api/update/groupe/{id} - Update a car group

    Example request:

    
    {
        "name": "Groupe Fiat!!"
    }

    Delete a Group

    DELETE /api/delete/groupe/{id} - Delete a Group
